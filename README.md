# 🏛️ Galeria

Bem-vindo ao **Galeria do Dinamiteink**, uma galeria de arte digital interativa e responsiva desenvolvida para exibir portraits exclusivos feitos inteiramente com **caneta esferográfica (Bic)**. Este projeto une a precisão do desenho manual com a modernidade do desenvolvimento web.

---

## 🎨 O Projeto
A aplicação foi construída com foco na experiência do usuário (UX), simulando o ambiente imersivo de uma galeria de arte física. O design utiliza a estética **Glassmorphism** (efeito de vidro) sobre o fundo clássico da "Noite Estrelada" de Van Gogh, criando um contraste único entre o clássico e o contemporâneo.

### ✨ Funcionalidades
- **Galeria Interativa:** Molduras com efeitos de hover e animações de entrada.
- **Visualização Detalhada (Modal):** Modal customizado com desfoque de fundo e detalhes técnicos da obra.
- **Interface Responsiva:** Adaptado para dispositivos móveis, tablets e desktops.
- **Integração com WhatsApp:** Botão de reserva que envia automaticamente o nome da obra para o artista.
- **Glassmorphism UI:** Uso avançado de `backdrop-filter` para uma interface premium.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **React** (com Vite)
* **TypeScript** (Tipagem estática para maior segurança do código)
* **CSS3** (Variáveis, Flexbox, CSS Grid e Glassmorphism)
* **Framer Motion** (Animações fluidas de interface)
* **Google Fonts** (Integração tipográfica clássica)
